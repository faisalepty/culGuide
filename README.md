# culGuide

An AI-powered cultural tourism guide that helps travelers discover local experiences, cultural activities, and navigate destinations with personalized recommendations based on their current location.

## Features

- **Location-Based Discovery**: Automatically detects your location and finds nearby cultural activities
- **Interactive Map**: Explore cultural activities with color-coded markers and real-time directions
- **AI Tourist Guide**: Chat with an intelligent assistant for location-aware recommendations
- **Cultural Activities**: Discover museums, performances, workshops, and local experiences near you
- **Smart Navigation**: Get directions from your current location to any cultural site
- **Festival & Events**: Stay updated on local cultural festivals and events in your area
- **Custom Itineraries**: Generate personalized travel plans based on your location and interests

## New Location Features

- **Automatic Location Detection**: Uses GPS to find your current position
- **Nearby Activity Search**: Finds cultural activities within your vicinity
- **Distance Information**: Shows how far each activity is from your location
- **Turn-by-Turn Directions**: Integrates with Google Maps for navigation
- **Location Status Indicator**: Shows when your location has been detected
- **Fallback Support**: Works with default locations if GPS is unavailable

## Getting Started

1. Open `index.html` in your web browser
2. Allow location access when prompted for personalized recommendations
3. Navigate through different sections using the bottom navigation
4. Interact with the map to explore cultural locations near you
5. Chat with the AI guide for location-specific recommendations

## Sections

- **Home**: Welcome screen with overview
- **Activities**: Browse cultural activities and attractions near your location
- **Map**: Interactive map with your location and nearby cultural markers
- **AI Guide**: Chat interface for location-aware personalized assistance

## Technical Implementation

- **Geolocation API**: For detecting user's current position
- **Leaflet Maps**: Interactive mapping with custom markers
- **Responsive Design**: Works on desktop and mobile devices
- **Location-Aware AI**: Provides contextual recommendations based on proximity
- **External Navigation**: Integrates with Google Maps for directions

## Files Structure

- `index.html` - Main application file with all functionality
- `styles.css` - Additional styling for enhanced UI
- `config.js` - Configuration settings for maps, APIs, and categories
- `README.md` - This documentation file
