# culGuide

An AI-powered cultural tourism guide that helps travelers discover local experiences, cultural activities, and navigate destinations with personalized recommendations.

## Features

- **Interactive Map**: Explore cultural activities with color-coded markers for different types of attractions
- **AI Tourist Guide**: Chat with an intelligent assistant for personalized recommendations
- **Cultural Activities**: Discover museums, performances, workshops, and local experiences
- **Smart Navigation**: Get directions and transportation information between locations
- **Festival & Events**: Stay updated on local cultural festivals and events
- **Custom Itineraries**: Generate personalized travel plans based on your interests

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through different sections using the bottom navigation
3. Interact with the map to explore cultural locations
4. Chat with the AI guide for personalized recommendations

## Sections

- **Home**: Welcome screen with overview
- **Activities**: Browse cultural activities and attractions
- **Map**: Interactive map with location markers
- **AI Guide**: Chat interface for personalized assistance

## Technology Stack

- HTML5, CSS3, JavaScript
- Leaflet.js for interactive maps
- Tailwind CSS for styling
- OpenStreetMap for map tiles

## Usage

The AI guide can help with:
- Finding cultural activities and attractions
- Restaurant and food recommendations
- Navigation and directions
- Festival and event information
- Creating custom itineraries

Simply type your questions in the AI Guide section or click on map markers for location-specific information.
